# AETERNUM

An interactive, fixed-viewport luxury jewelry concept: a digital Pantheon navigated through spatial halls rather than a conventional store layout.

Open `index.html` directly, or serve the folder with any static server.

## UX Case Study

- Open `case-study.html` for the separate UX case study page.
- The case study supports Russian, Polish, Ukrainian, and English through the language switcher.

## Interaction

- Enter the Pantheon and choose one of seven collection arches.
- Use the rotunda arrows, keyboard arrow keys, or the plan indicator.
- Select jewelry pedestals for a private museum-style product view.
- Ambient sound is optional and off by default.
- Mobile switches to a vertical temple journey for comfortable touch navigation.
